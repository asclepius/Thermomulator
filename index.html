Thermomulator by Paul Kulyk

An arduino based simulation thermometer for medical simulation.

This page hosts the instructions for the design and implementation of the Thermomulator medical simulation thermometer. This will include detailed photographs of the assembly of the project as well as the source code used for the arduino.
